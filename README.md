# XO-Battle: Tic Tac Toe Game

A web-based Tic Tac Toe game built with Flask, featuring a computer opponent with strategic gameplay.

## Features

- Play against a computer opponent
- Choose your symbol (X or O)
- Computer makes strategic moves
- Score tracking
- Responsive design
- Modern UI with custom color scheme

## Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript

## Setup Instructions

1. Clone the repository:
```bash
git clone https://github.com/yourusername/xobattle.git
cd xobattle
```

2. Create a virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Run the application:
```bash
python app.py
```

5. Open your browser and navigate to:
```
http://localhost:5000
```
